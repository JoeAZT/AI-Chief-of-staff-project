# AI Chief of Staff

An AI system that plans your day, holds you accountable, and calls you out when you're avoiding things.

Not a to-do app. Not a calendar. A coach that wakes up before you, looks at your goals, tells you exactly what to do today — then checks in at night and asks what happened.

Built on [Claude Code](https://docs.anthropic.com/en/docs/claude-code). Runs in your terminal. Writes to markdown files. Syncs to your phone via Apple Calendar.

## What it does

**Morning briefing** (runs daily at 9am or on demand)
- Reads your to-do list, calendar, and project statuses
- Generates 3–5 specific, actionable tasks for today
- Proposes a schedule fitted to your available hours
- Suggests a workout and learning content with direct links
- Flags stuck tasks: 2 days → nudge, 3 days → escalation, 5 days → intervention
- Creates Apple Calendar events with context notes (syncs to your phone)

**Evening review** (runs daily at 9pm or on demand)
- Compares what was planned vs what got done
- Spots patterns — are you always skipping fitness? Avoiding a specific task?
- Checks if your workout actually happened (reads the log, not the plan)
- Asks one reflection question to think about before bed

**Weekly review** (runs Sundays at 10am or on demand)
- Completion rate across the week
- Productivity patterns — which time slots work, which don't
- Goal progress — are you actually getting closer?
- One thing to do differently next week

## How it works

Everything is personalised. When you run setup, the system asks you questions about your schedule, goals, energy patterns, fitness setup, and accountability preferences. It generates:

- A `CLAUDE.md` profile tuned to you
- A workout library matched to your equipment
- A config file with your paths and calendar
- A starter to-do list

The system then uses these files every time it runs, so the briefings are specific to your life — not generic productivity advice.

## Prerequisites

- macOS (Apple Calendar integration uses AppleScript)
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) installed (`brew install claude-code` or via npm)
- A folder for briefing output (Obsidian vault, Notes folder, or any directory)
- Apple Calendar with your Google/iCloud account synced (for phone sync)

## Setup

```bash
git clone https://github.com/YOUR_USERNAME/ai-chief-of-staff.git
cd ai-chief-of-staff
bash setup.sh
```

The setup takes ~5 minutes. It asks you questions one at a time and generates your personalised system. No forms, no config files to edit manually.

## Usage

### Daily commands

```bash
# Morning briefing — generates your daily plan
bash morning-briefing.sh

# Approve today's schedule — review and add events to Apple Calendar
bash approve-schedule.sh

# Log a completed workout
bash log-workout.sh "C — Full Body Power"

# Evening review — what got done, what didn't, prep for tomorrow
bash evening-review.sh

# Weekly review — patterns, progress, accountability
bash weekly-review.sh
```

### Automate it (optional)

Set up macOS LaunchAgents to run the briefing and review automatically:

**Morning briefing at 9am** — save as `~/Library/LaunchAgents/com.cos.morning.plist`:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.cos.morning</string>
    <key>ProgramArguments</key>
    <array>
        <string>/bin/bash</string>
        <string>/Users/YOUR_USERNAME/ai-chief-of-staff/morning-briefing.sh</string>
    </array>
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>9</integer>
        <key>Minute</key>
        <integer>0</integer>
    </dict>
</dict>
</plist>
```

Load it: `launchctl load ~/Library/LaunchAgents/com.cos.morning.plist`

Repeat for evening review (9pm) and weekly review (Sunday 10am), changing the label, script path, and time.

## How the accountability works

The system doesn't let tasks quietly rot on your list:

| Days stuck | What happens |
|------------|-------------|
| 2 days | Flagged: "Is something blocking you?" |
| 3 days | Escalated: suggests breaking it down, rescheduling, delegating, or dropping it |
| 5 days | Dedicated section: "We need to talk about this." Diagnoses avoidance, skill gaps, or wrong priorities |

The evening review also detects when the loop itself breaks — if you stop running the evening review for 3+ days, the morning briefing warns that it's "planning blind."

## What's in the box

| File | Purpose |
|------|---------|
| `setup.sh` | Interactive onboarding — generates your personalised system |
| `onboarding-prompt.md` | System prompt that guides the onboarding conversation |
| `CLAUDE.md` | Your personalised profile, goals, schedule, and rules (generated by setup) |
| `config.sh` | File paths and calendar name (generated by setup) |
| `workout-library.md` | Workout routines matched to your equipment (generated by setup) |
| `morning-briefing.sh` | Collects context and generates the daily briefing |
| `evening-review.sh` | Generates the end-of-day review |
| `weekly-review.sh` | Generates the weekly performance review |
| `create-events.sh` | Creates Apple Calendar events from the proposed schedule |
| `approve-schedule.sh` | Review and approve the schedule before it hits your calendar |
| `log-workout.sh` | Log a completed workout to the fitness log |
| `get-calendar.scpt` | AppleScript to read today's calendar events |

## Customisation

Everything is in markdown and shell scripts — no hidden config.

- **Change your goals or schedule:** Edit `CLAUDE.md` directly
- **Change your calendar or file paths:** Edit `config.sh`
- **Change your workouts:** Edit `workout-library.md`
- **Re-run onboarding from scratch:** Delete `CLAUDE.md`, `config.sh`, and `workout-library.md`, then run `bash setup.sh` again
